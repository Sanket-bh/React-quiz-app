## React Quiz App

This is a simple quiz application built while learning the basics of React.
The project focuses on understanding React components, state management,
and user interaction.

### What This Project Covers
- Creating reusable React components
- Managing state using React hooks
- Handling user input and events
- Displaying quiz questions and tracking scores

### Features
- Multiple-choice quiz questions
- Score calculation at the end of the quiz
- Simple and clean user interface

### Tech Stack
- React
- JavaScript
- HTML
- CSS

### Purpose
This project was created as part of my React learning journey to understand
how frontend applications work and how user interactions are handled in React.
